# Ubik brand asset slots

Replace these files when the final SVGs are ready:

- `logo.svg` — default/light mark, square viewport recommended.
- `logo-dark.svg` — dark-mode mark.
- `logo-wordmark.svg` — optional horizontal lockup for light mode.
- `logo-wordmark-dark.svg` — optional horizontal lockup for dark mode.

The current static site maps these paths through `/brand-assets.js`, so logo filenames only need to change in one place if the final assets use different names.
